# Life-Table
Life Table project

Link to project - https://docs.google.com/spreadsheets/d/15qFrDGDchRUf2r1a_5xfxE8dqtwWEUt5av2AuEN6TaA/edit?gid=1646675705#gid=1646675705

Data from Social Security Administration - https://www.ssa.gov/oact/STATS/table4c6.html

A life table is a tool used to display a summary of mortality and surival rates for a population. It calculates the probability a person of a certain age will die or survive and can show expected future life expectancy. This is a life table I created based on the Social Security Administration's data on the US population separated by male and female. I then used it to build a survivorship curve to graphically represent the relationship between age and mortality, as well as a simple bar chart to show the number of projected deaths at each age. Lastly, there is also a mortality adjustment factor that can be used to manipulate the mortality rate of the population and compare the results.
